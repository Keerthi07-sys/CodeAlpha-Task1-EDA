# CodeAlpha Task 1 - Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a sales dataset using Python.

## Features
- Data inspection
- Data cleaning
- Summary statistics
- Missing value detection
- Correlation analysis
- Data visualization

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Files Included
- eda_analysis.py
- sales_data.csv

## How to Run
1. Install the required libraries:
   pip install pandas matplotlib seaborn

2. Run the project:
   python eda_analysis.py

## Output
The project generates statistical summaries and visualizations to identify trends, patterns, and insights from the dataset.
